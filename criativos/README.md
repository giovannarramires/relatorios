# Criativos para anúncios Meta

Ponte para subir imagens de anúncio sem passar pelo Gerenciador.

As imagens colocadas aqui viram URL pública que o Meta consegue baixar
(usar a raw — responde na hora; a de Pages leva ~1min p/ buildar):

- `https://raw.githubusercontent.com/giovannarramires/relatorios/main/criativos/<cliente>/<arquivo>.jpg`
- alt: `https://relatorios.giogrowthdigital.com.br/criativos/<cliente>/<arquivo>.jpg`

Usado pela skill `subir-campanha-meta`. Nomes sem espaços/acentos.

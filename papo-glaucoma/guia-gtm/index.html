Instalação do Google Tag Manager no app do quiz

Guia para adicionar o container GTM ao aplicativo do quiz do Papo Glaucoma (/estagio). Toda a configuração de tags já está pronta e publicada — falta apenas este passo no código do app.

Container: GTM-MKRDH4M App: papoglaucoma.com.br/estagio Tempo: ~10 min + deploy

O que é preciso fazer: inserir o script do Google Tag Manager (container GTM-MKRDH4M) no aplicativo do quiz. É uma edição única no layout raiz do app.

Por que: o rastreamento de conversões (Meta Pixel e Google Analytics) já está montado e publicado no GTM, mas o app do quiz ainda não carrega o container. Sem esta instalação, nenhum evento é registrado.

Caminho recomendado — Next.js (App Router)

Usa o pacote oficial do Next.js, que injeta o script e o <noscript> automaticamente. É o caminho mais simples e seguro.

Alternativas (somente se não usar o pacote oficial)

Escolha uma das opções abaixo conforme a estrutura do projeto. Não é necessário aplicar mais de uma.

Opção A — App Router, manualmente

Se o projeto usa a pasta app/ e você prefere não instalar o pacote.

Opção B — Pages Router

Se o projeto usa a pasta pages/ (estrutura mais antiga).

⚠ Ponto de atenção

O quiz em /estagio parece ser um aplicativo separado do site principal. O site principal (a home) já tem o GTM instalado, mas o app do quiz não o herda automaticamente. Por isso, o código acima precisa ser inserido no layout raiz do próprio app do quiz — não no site institucional. Use sempre o ID GTM-MKRDH4M.

Checklist de validação

  • O código foi adicionado ao layout raiz do app do quiz (não ao site principal).
  • O ID usado é exatamente GTM-MKRDH4M.
  • O deploy foi concluído e a nova versão está no ar.
  • No console de /estagio/quiz, window.google_tag_manager retorna um objeto com GTM-MKRDH4M.
  • Avisar a equipe da Gio Growth Digital para validar os eventos no modo de teste do GTM.

Assim que a instalação estiver no ar, a Gio Growth Digital roda o modo de pré-visualização (Tag Assistant) do GTM e confirma que os eventos de conversão do quiz — Lead, etapas e clique no WhatsApp — estão disparando corretamente. Nenhuma outra ação de rastreamento é necessária do lado do desenvolvimento.
